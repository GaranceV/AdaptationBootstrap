# AdaptationBootstrap
Adaptation of HMI project, based on bootstrap responsive design


## Lancement
Pour lancer le projet, la première fois : 
`npm install`
`bower install`
Suite à cela, et toutes les autres fois : 
`grunt serve`

Le projet va alors se lancer sur localhost, sur le port 1818. 
On peut modifier ce paramètre dans gruntfile.js, à la racine du projet. 

## Responsive design
Ce projet est prévu pour s'adapter aux différents usage qu'on peut faire d'une page contenant une recette de cuisine. 
Si vous le regarder au format mobile, vous ne verrez que la liste des ingrédients. 
Au format tablette, on ne voit plus que les étapes de la recette
Au format bureau, on voit la liste des ingrédients et les étapes, en vue d'ensemble.
